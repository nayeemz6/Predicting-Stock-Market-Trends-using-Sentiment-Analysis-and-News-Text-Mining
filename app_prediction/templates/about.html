About This Project

Welcome to the Stock Prediction and Sentiment Analysis platform, an academic dissertation project.

Research Objectives

This project aims to explore the applications of machine learning algorithms and natural language processing techniques in financial markets. Specifically, it focuses on predicting stock prices and performing sentiment analysis on headlines related to the stocks.

Methodology

The system uses Long Short-Term Memory (LSTM) networks, a type of recurrent neural network, to make stock predictions based on historical data. For sentiment analysis, it employs the VADER sentiment analysis tool and topic modeling.

Academic Context

This dissertation project serves as a contribution to the ongoing research in the fields of machine learning, natural language processing, and finance. It aims to demonstrate the effectiveness and limitations of using advanced algorithms for stock prediction and sentiment analysis.

Disclaimer

Please note that while this platform aims to provide accurate predictions, it is part of an academic project and should not be used for making real-world investment decisions.
